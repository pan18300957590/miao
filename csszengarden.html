Requirements

Where possible,we would to see mostly CSS 1 & 2 usage. CSS 3 & 4should be limited to widely-supported elements only, or strong fallbacks should be provided. The CSS Zen Garden is about functional, practicalCSSand not the latest bleeding-edge tricks viewable by 2% of the browsing public. The only real requirement we have is that you CSSvalidates.

Luckily,designing this way shows how well various browsers have implementedCSSby now.When sticking to the guidelines you should see fairly consisient results across most mofern browsers. Due to the sheer number of use agents on the web these days — especially when you factir in movile — pixel-perfect layouts may not be possible across evety platfirm.
